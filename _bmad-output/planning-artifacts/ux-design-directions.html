Landing Page

The first thing users see when they tap the link from group chat. One job: enter a username and get to the bracket.

Approach: Nothing but the essentials. Logo, subtitle, input, button. Full-screen centered layout creates a focused moment. Zero distraction. The user's eye goes directly to the input field.
Strengths
Maximum clarity. Impossible to get confused. Loads instantly. Works on any screen size without adaptation.
Tradeoffs
No context about the pool status (how many have entered, is bracket locked, etc.). Could feel sparse.
